Instructions

This interface aims to facilitate the interaction with a chatbot that ingests structured data in the form of SPO triples

On every turn you will be asked to fill in a form to represent the structured knowledge in your utterance. The form will be prefilled with suggestions for certain fields. You can always change the fields manually, if you'd prefer.

Please focus on filling the triple (subject, predicate, object fields) and perspective (certainty, polarity, sentiment) parts of the form. As a secondary goal, you could input the utterance in natural language, but this is not strictly necessary.

Certainty values should range from 0 (uncertain) to 1 (certain).

Polarity values may be 1 (affirmative) or -1 (negation).

Sentiment values may be 1 (positive), -1 (negative), or 0 (neutral).

Submit as many capsules as you wish in a chat. When want to stop the chat, finish session via the top menu bar.

Notes

Please pay attention to be use lowercase wherever possible. Only submit statements, no questions
